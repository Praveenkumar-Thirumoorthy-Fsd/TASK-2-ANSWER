1.Difference between document and window objects
  Document oject :
  The document windows is the area where all pages are created and shapes can be positioned.
  During both the editing and play modes the document window consists of two distinct panels vertically aligned
  With the upper large panel being the area for the currently displayed page.

  window ojects :
  The windows object represents a window/tab containing a DOM document where as document object is property of window object that points DOM document in that window
